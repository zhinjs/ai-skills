# AI Agent Guidance

This repository contains Zhin-focused AI skills and best practice references. Use this file for short, high-level expectations for AI agents working with this repo.

## Goals

- Prefer minimal, targeted changes.
- Keep skills aligned with the Agent Skills specification.
- Base examples on official Zhin sources (core package + examples).
- Avoid adding new tooling or dependencies unless required.

## Structure

- `skills/` contains Agent Skills (`SKILL.md`).
- `packages/zhinjs-best-practices/` contains human-readable best practice references.

## When Updating Skills

- Ensure each skill name matches its directory name.
- Provide actionable instructions and examples.
- Keep content focused and avoid unrelated guidance.
